# Assignment 4 – Machine Learning

This repository contains solutions for Assignment 4.

## Files
- assign4q1.py – Explore Breast Cancer dataset and analyze class distribution
- assign4q2.py – Train Decision Tree using entropy and evaluate accuracy

- assign4q5.py – Compare Decision Tree and Neural Network using confusion matrices
